<!doctype html> : 웹페이지가 어떤 버전의 HTML 언어로 작성되었는지 선언
<html></html> : 웹페이지의 시작과 끝
<title></title> : 웹페이지의 제목 지정
<head></head> : 본문을 설명하는 태그
<meta charset="utf-8"> : utf-8로 웹페이지를 나타내라 (한글)
<body></body> : 본문 태그
<h1></h1> : 제목 태그
<a href="링크"></a> : 링크 태그
	target="_blank" : 새 탭으로 열어라
	title="...." : 링크 설명
<li></li> : 목록 태그 (반드시 부모 태그를 가지고 있다)
<ul></ul> : 목록 태그의 부모 (반드시 자식 태그를 가지고 있다)
<ol></ol> : 목록 넘버링 태그
<strong></strong> : 글씨 진하게 표시
<u></u> : 밑줄
<br> : 줄바꿈
<p></p> : 단락을 나타내는 태그
<p style="margin-top:40px;"></p> : 단락 위쪽에 여백 주기
<img src="이미지 이름" width="100%"> : 이미지 첨부
유튜브 -> 공유 -> 퍼가기 -> 코드 복사 -> 붙여넣기 : 동영상 첨부

    = border: 5px solid red;
    display:inline; : 자기 자신의 부피만큼 쓰도록 하기

    width: 100px ; : 폭 길이 조정

<style="color:red"> : 지정된 곳의 글자색을 빨강으로 바꿔라
<text-decoration:underline" : 지정된 곳의 글자에 밑줄을 쳐라
<class="saw"> : 같은 그룹으로 묶기
<class="saw active"> : 여러 개의 값이 들어올 수 있으며, 띄어쓰기로 구분한다
<id="active">
화면 전체를 쓰는 태그 : block level element
자기 자신의 부피만큼 쓰는 태그 : inline element
